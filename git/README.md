# README

这不是一个初级git教程，主要是我在用git和github的时候遇到了一些麻烦，记录一些更复杂的问题，来源主要是参考pro git，初学git建议廖雪峰就OK了。

git命令辣么多，我们来给你捋一捋哈

第1组：
对应[1.本地分支](./1.本地分支.md)中的1.1
```shell
git init
git add
git commit
git reset
```

会了第一组你就玩git了。

接下来，你有两种选择，第一种开始学习远程仓库，学会使用github。第二种就是继续玩转本地分支。

第二组：远程仓库(这里只列基础的，fetch在进阶哈)
对应[2.远程分支](./2.远程分支.md)中的2.1和2.1
```shell
git remote add
git push
git clone
```

第三组：继续玩转本地分支
对应[1.本地分支](1.本地分支.md)中的1.2
```shell
git branch
git checkout
git merge
```
